# Quickstart: Capabilities Page

This guide outlines how to view and update the Capabilities Contract document in the application.

## Storing the Markdown Document

1. The Markdown file containing all capabilities and features is located at:
   `src/assets/capabilities.md`
2. If you need to edit descriptions, add features, or update terms for a client meeting, edit this file directly.

## Viewing in the Application

1. Run the local development server:
   ```bash
   npm run dev
   ```
2. Log in with an admin or instructor account.
3. Click the **Capabilities** link in the left sidebar menu.
4. The application will render the Markdown document directly on the page, matching the application's dark mode and typography.
