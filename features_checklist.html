Application Features Checklist — Client Verification —

1. Authentication & Security

Sign In Page

Email and password login form
Show/hide password toggle
Form validation (empty fields, format errors)
Deactivated account error message
"Forgot Password?" navigation link
"Register" navigation link

Registration Page

Full Name, Email, Phone inputs
Password and Confirm Password with match validation
Redirect to login on success

Forgot Password Page

Email input to trigger reset link
Confirmation success message

Reset Password Page

Token validation from URL
New password and confirm password inputs
Redirect to login on success

2. Main Dashboard

Quick Actions Panel

"Enroll Student" quick link
"Search Directory" quick link
"Create Receipt" quick link
Today's total session count display

Day Selector Bar

7-day horizontal calendar bar
Highlights selected date
Filters session list by selected date

Instructor Selector Bar

Horizontal bar of instructors scheduled for the day
Filters session list by selected instructor
"All Instructors" reset option

Desktop Session Cards & Attendance Sheet

Groups scheduled for the selected day are listed
Displays: group name, course, instructor, time, level/session progress
Expandable student roster with attendance checkboxes
Real-time attendance save (Present / Absent)

Mobile Dashboard

Bottom navigation bar replaces sidebar on mobile
Mobile-optimized day and instructor selectors
Compact mobile group cards
"Check Attendance" button opens bottom-sheet attendance drawer
Floating Action Button (FAB) for quick actions

Directory Tabs

Students tab
Parents tab
Waiting List tab
Advanced Search tab (multi-variable filters)

Global Actions & Filters

Main search bar (name, phone, email)
A–Z Alphabet slider filter bar
Group-by dropdown (Grade, School, etc.)
Add Student button & modal
Inputs: Name, Date of Birth, Gender, School, Grade, Notes
Parent search and link selector
Add Parent button & modal
Inputs: Name, Phone Primary, Phone Secondary, WhatsApp, Email, Address, Relationship
Trash bin toggle (show soft-deleted records)
Restore soft-deleted record
Permanently delete record

Student Detail Page

Identity header card (name, status badge, grade, school, registration date)
Parent/guardian card with link, edit, and unlink options
Siblings connection panel
Active enrollments overview (course, level, group, price, discount)
Quick links to modify or drop enrollment
Competitions & teams tracker with placement ranks
Invoices & payment history grid (balance, paid, outstanding)
Activity / audit log timeline

Parent Detail Page

Contact details card (phones, email, WhatsApp, address)
Connected children list with navigation links
"Add Child" quick association button

4. Academic Course Catalog

Course Index Page

Table and Card view toggle
Search bar (title, category)
Pagination (10 / 20 / 50 per page)
Column sorting (ID, Title, Category, Price, Sessions)
Create Course modal
Inputs: Title, Category, Price per Level, Sessions per Level, Description, Status
Edit Course modal (pre-filled)
Delete Course dialog (blocked if active groups exist)

Course Detail Page

Stats strip (groups count, enrolled students, attendance rate)
Details tab (duration, fees, status, description)
Groups tab (lists groups with name, instructor, level, capacity, status)
Navigate to group detail from list

5. Academic Groups & Rosters

Groups Index Page

Table and Card view toggle
Search bar (group name)
Group By selector (Day / Instructor / Course / Status)
Advanced filter panel
Multiselect: Course, Instructor, Day, Level, Status
Active filter tags with remove buttons and Clear All
Pagination (10 / 20 / 50 / 100 per page)
Create Group modal
Inputs: Name, Course, Instructor, Capacity, Start Date, Schedule (Day / Start / End time)
Edit Group modal (updates schedule, instructor, capacity)

Group Detail Page

Group header (status badge, instructor, course, capacity, active level)
Edit Group action
Archive / Unarchive Group action
Delete Group action
Level Up button (instant progression to next level)
Progress Level dialog (advanced: change name, instructor, course, price override)
Notes field with auto-save

Levels & Sessions Tab

All historical levels listed
Session attendance log per level (number, date, time, rate)
Level payments overview (due vs paid vs outstanding per student)
Add student to roster (combobox search)
Remove student from roster (blocked if payment exists)
Collect payment for student in roster
Amount input (supports partial payments)
Optional parent link search
Updates balance, logs invoice, emails receipt
Payment status badges: Paid / Partial / Pending

History Tab

Chronological log of roster additions, drops, and graduations

6. Enrollments Management

New Enrollment Panel

Student selector (searchable combobox)
Course selector
Academic Level selector
Group selector (filtered by course/level with open seats)
Discount input (flat EGP or percentage)
Pricing breakdown (standard price → discount → final amount)
Payment method selector (Cash / Card / Installment)
Submit: creates enrollment, assigns to group, generates invoice

Modify Enrollment Panel

Select existing student enrollment
Adjust discount (flat or percentage)
Override price field
Live billing breakdown (original, paid, new due, remaining)
Notes / reason text input (saved in audit log)
Blocks save if result is negative balance
Auto-sends email update to parent

Drop Enrollment Panel

Select enrollment to drop
Option: forgive or keep outstanding balance
Drop date selector and reason text area
De-registers student from group and locks enrollment as "Dropped"

Table of all receipts processed today

View PDF receipt in new tab

Create Receipt Tab

Student selector (searchable)
Parent selector (auto-linked or searchable)
Lists student's enrollments with unpaid balances
Amount paid input (full or partial)
Payment method: Cash / Card / Bank Transfer / Instapay
Submit: updates balance, logs receipt, emails parent

Unpaid Enrollments Tab

All students with outstanding balances
Columns: Student, Parent contact, Course, Group, Total, Paid, Balance Due
"Collect Payment" button (pre-fills Create Receipt)

Refunds Tab

Refund records panel
